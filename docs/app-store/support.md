# ClipDock Support / 技术支持

## Quick Start / 快速开始

1. Open ClipDock.
2. Tap **Choose External Folder** and select a destination folder (it can be on an external drive via the Files app).
3. Tap **Scan Videos**.
4. Select the videos you want to migrate.
5. Tap **Start Migration**.
6. After migration completes, you can optionally delete originals (only for items that were migrated and validated successfully).

1. 打开 ClipDock。
2. 点击 **选择外接目录** 并选择目标文件夹（可通过“文件”App 选择外接盘目录）。
3. 点击 **扫描视频**。
4. 勾选要迁移的视频。
5. 点击 **开始迁移**。
6. 迁移完成后，你可以选择删除原件（仅对“迁移并校验成功”的条目开放删除）。

## Troubleshooting / 常见问题

### A) Migration fails with permission error / 迁移提示无权限

- Reconnect your external drive.
- In ClipDock, tap **Recheck Folder Access**.
- If it still fails, tap **Choose External Folder** and select the destination folder again.

- 重新插拔外接存储。
- 在 ClipDock 点击 **重新检查目录权限**。
- 若仍失败，请点击 **选择外接目录** 重新选择目标文件夹。

### B) Video size shows "--" / 视频大小显示 "--"

ClipDock only shows best-effort local sizes without downloading iCloud-only assets. If a video is not stored locally, size may show as "--" until it is downloaded/exported.

ClipDock 只在不触发 iCloud 下载的前提下尽力显示本地大小；若视频不在本地，大小可能显示为 "--"，直到该视频被下载/导出。
